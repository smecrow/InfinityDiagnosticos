# Stack Definida

Este arquivo registra a stack escolhida para o projeto.

## Frontend

- Next.js
- TypeScript

## Backend

- Java
- Spring Boot
- Spring Security

## Banco de dados

- PostgreSQL

## Observacoes

- O backend sera responsavel pela autenticacao, autorizacao, persistencia dos diagnosticos e painel administrativo.
- O frontend sera responsavel pela interface do cliente e pelas telas administrativas.
- Esta definicao de stack serve como base para os proximos passos de arquitetura e implementacao.

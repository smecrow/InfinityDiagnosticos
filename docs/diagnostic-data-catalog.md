# Catalogo de Dados do Diagnostico

Este arquivo registra os dados que devem ser coletados no website de testes de conexao.

## Relacionado ao dispositivo

- Tipo de dispositivo
- Sistema operacional
- Navegador e versao
- Idioma
- Fuso horario
- Plataforma reportada
- Quantidade de nucleos logicos da CPU
- Memoria RAM

## Relacionado a conexao

- Latencia ate o servidor
- Jitter
- Perda de pacotes
- Velocidade de download
- Velocidade de upload
- Tipo de conexao reportado pelo navegador
- Status online / offline

## Observacoes

- Esta lista define os dados desejados para o projeto neste momento.
- A implementacao deve validar depois quais campos estao disponiveis de forma consistente entre navegadores.
